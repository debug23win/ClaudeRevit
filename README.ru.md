# Claude Revit

[English](README.md) | **Русский**

Claude AI внутри Autodesk Revit 2027 — закрепляемая панель чата со **131 инструментом**, через которые Claude напрямую читает и изменяет вашу модель. Попросите его построить стены, создать спецификации, расставить семейства, образмерить оси, заармировать конструкции, оформить листы — и он сделает это сам.

---

## Возможности

- **Закрепляемая панель чата** в Revit с потоковым выводом ответов
- **131 инструмент**: моделирование, виды, листы, аннотации, спецификации, фильтры, семейства, армирование
- **Одна отмена на запрос** — Ctrl+Z откатывает всё, что Claude сделал за один ход
- **Понимает выделение** — зелёная плашка показывает, что выделено; Claude знает, что значит «это»
- **Markdown** и **выделяемый текст** в сообщениях
- **Кликабельные ID элементов** — клик по ID в результате инструмента выделяет и показывает элемент в модели
- **Учёт стоимости** с кэшированием промпта (TTL 1 час на системный промпт, инструменты и историю диалога — каждый ход оплачивает только новые токены)
- **Полная память контекста** — вся история диалога (включая вызовы инструментов, результаты и ID элементов) переживает перезапуск Revit: Claude помнит, что он построил
- **Автокомпакция** — при превышении ~120K токенов старые ходы сжимаются в резюме (через Haiku), не переполняя контекстное окно
- **Самообучение** — инструмент `save_memory`: Claude сохраняет ваши предпочтения и стандарты проекта и применяет их во всех будущих сессиях
- **`execute_csharp`** — доступ ко всему Revit API: когда готового инструмента нет, Claude пишет C#-код, который компилируется (Roslyn) и выполняется в транзакции — после вашего явного разрешения
- **Подтверждение опасных операций** — удаление элементов и выполнение кода требуют кнопки «Да» в диалоге
- **Выбор модели** — Sonnet 5 / Opus 4.8 / Fable 5 / Haiku 4.5 (+ Sonnet 4.6, Opus 4.7)
- **Enter — отправить**, Shift+Enter — новая строка
- **API-ключ вводится в панели** (шестерёнка) и хранится зашифрованным (Windows DPAPI) — без открытых переменных окружения

## Установка (для пользователей)

Понадобятся:

- **Autodesk Revit 2027**
- **Windows** (Revit работает только на Windows)
- **API-ключ Anthropic** — получите на [console.anthropic.com](https://console.anthropic.com/settings/keys) и пополните баланс в **Billing**

Любой из двух способов:

### Вариант А — установщик .exe (проще всего)

Скачайте **`ClaudeRevit-Setup-vX.Y.exe`** из [последнего релиза](https://github.com/debug23win/ClaudeRevit/releases/latest), запустите, Next → Install. Готово.

> Windows SmartScreen при первом запуске может показать «Windows защитила ваш компьютер» (установщик пока не подписан). Нажмите **Подробнее → Выполнить в любом случае**.

### Вариант Б — одна команда PowerShell

```powershell
iwr https://raw.githubusercontent.com/debug23win/ClaudeRevit/main/install.ps1 | iex
```

В обоих случаях: запустите Revit, откройте любой проект и найдите вкладку **Claude** в ленте. Кнопка **Chat** открывает панель справа. При первом запуске нажмите **⚙** и вставьте API-ключ.

Для обновления просто повторите установку — подтянется последний релиз.

## Сборка из исходников (для разработчиков)

Понадобятся:

- **Visual Studio 2026 Community** (или Rider, или VS Code с C# Dev Kit)
- **.NET 10 SDK** ([скачать](https://dotnet.microsoft.com/download/dotnet/10.0))
- **Autodesk Revit 2027** локально (нужен только для отладки по F5 — компиляция работает и без него)

Шаги:

1. Клонировать:
   ```powershell
   git clone https://github.com/debug23win/ClaudeRevit.git
   cd ClaudeRevit
   ```
2. Открыть `ClaudeRevit.sln` в Visual Studio.
3. ПКМ по проекту → **Restore NuGet Packages**.
4. Назначить `ClaudeRevit` стартовым проектом.
5. Нажать **F5**.

Пост-сборочный шаг сам копирует DLL и манифест в `%AppData%\Autodesk\Revit\Addins\2027\`. F5 запускает Revit с подключённым отладчиком.

### Как устроена сборка

- Ссылки на Revit API берутся из NuGet-пакетов **Nice3point.Revit.Api.RevitAPI** и **RevitAPIUI** — локальный Revit для компиляции не нужен.
- Пост-сборочный таргет `DeployToRevit` копирует в `%AppData%\Autodesk\Revit\Addins\2027\`. Чтобы пропустить (например, на CI): `-p:SkipDeploy=true`.
- Отдельный таргет `PackageRelease` складывает артефакты релиза в `bin\Release\release\` — его использует GitHub Actions.

## Другая версия Revit

Поменяйте `<RevitVersion>2027</RevitVersion>` в `ClaudeRevit/ClaudeRevit.csproj` на нужную (например, `2028`). Пакеты Nice3point используют шаблон версии (`$(RevitVersion).0.*`) и подхватят соответствующий API автоматически — если такой пакет существует.

## Инструменты

Плагин даёт Claude **131 инструмент** по категориям:

- **Чтение модели** — элементы, параметры, уровни, материалы, стадии, семейства, информация о проекте, предупреждения
- **Создание геометрии** — стены, перекрытия, крыши, помещения, уровни, оси, двери, окна, колонны, балки, фундаменты, MEP (воздуховоды/трубы)
- **Армирование** — ряды стержней (с количеством и шагом), площадное (сеточное) армирование стен и плит, список типов арматуры, инспекция существующего армирования
- **Самообучение и запасной выход** — `save_memory` (постоянные заметки, которые Claude применяет во всех будущих сессиях) и `execute_csharp` (произвольный код Revit API с подтверждением Allow/Deny)
- **Операции с элементами** — перемещение, поворот, копирование, зеркало, массив, удаление, параметры, закрепление
- **Виды** — 3D, планы, разрезы, фасады, фрагменты, дублирование, масштаб, шаблоны видов
- **Листы** — создание листов, размещение видов и спецификаций
- **Спецификации** — настоящие ViewSchedule с выбором полей
- **Аннотации** — размеры, марки, текст, линии детализации, области штриховки, опорные плоскости
- **Видимость** — скрыть/изолировать, цветовые переопределения, фильтры
- **Интерактив** — `pick_point_in_view` для сценариев «кликни, куда поставить»
- **Семейства** — загрузка .rfa, список загруженных, размещение экземпляров
- **Группы** — создание и размещение
- **Экспорт** — изображения видов, PDF, DWG, CSV спецификаций, сохранение документа

Полный список — в папке [`ClaudeRevit/Tools/`](ClaudeRevit/Tools): один `.cs`-файл = один инструмент.

## Архитектура

- **`App.cs`** — точка входа `IExternalApplication`: регистрирует инструменты, панель и кнопку в ленте
- **`Tools/`** — классы `IRevitTool`. Новый файл + строка регистрации в `App.cs` → инструмент доступен Claude
- **`Tools/ToolDispatcher.cs`** — `IExternalEventHandler`: исполняет вызовы в API-потоке Revit; каждый инструмент — в `Transaction`, весь ход — в `TransactionGroup` (одна отмена на запрос)
- **`Services/AnthropicChatService.cs`** — агентский цикл: стриминг, вызовы инструментов, история с thinking-блоками, кэширование промпта, компакция
- **`Services/HistoryStore.cs` / `MemoryStore.cs` / `ApiKeyStore.cs`** — персистентность: история диалога, память Claude, зашифрованный ключ
- **`UI/ChatPaneView.xaml`** — WPF-панель чата (markdown, кликабельные ID, подтверждения операций)
- **`Services/SelectionService.cs`** — отслеживание выделения через событие `Idling`
